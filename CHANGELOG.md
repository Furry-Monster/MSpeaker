# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-30

### Added

- Initial release of MSpeaker dialogue system
- Custom dialogue script format (.msp files)
- Flexible dialogue engine with base and simple implementations
- Advanced control flow support (conditions, loops, variables, functions)
- Extensible UI view system with base view and typewriter effect view
- Complete choice system with button management and dialogue branching
- Global variables and function system
- Resource management via MspDialogueAsset ScriptableObject
- Editor tools including script importer and preview window
- Plugin system for extended functionality
- Comprehensive test suite
- Sample project with example dialogue and custom functions
